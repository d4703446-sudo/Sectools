## 📌 О проекте

SecTools — веб-приложение с набором инструментов для изучения основ информационной безопасности. Все вычисления выполняются локально в браузере — данные не передаются на сервер.

---

## 🛠 Инструменты

| Инструмент | Описание |
|---|---|
| 🔑 Генератор паролей | Криптографически стойкая генерация через `window.crypto.getRandomValues()` |
| 🛡 Анализ пароля | Оценка надёжности + расчёт времени brute-force атаки |
| 🔒 Шифрование | Цезарь, Виженер, XOR, Base64 |
| 🔢 Конвертер СС | Перевод между двоичной, восьмеричной, десятичной, hex + таблица ASCII |
