SecTools

Набор инструментов для изучения основ информационной безопасности: генерация паролей, шифрование текста, анализ надёжности и конвертация систем счисления.

🔑 Генератор паролей
Результат
—
Как это работает: генератор формирует случайный пароль из выбранного набора символов с помощью window.crypto.getRandomValues() — криптографически стойкого генератора случайных чисел браузера.

🔒 Шифрование текста
Base64 — кодирование, не шифрование. Ключ не используется.
Результат
—
Шифр Цезаря — один из древнейших шифров подстановки. Каждая буква сдвигается на фиксированное число позиций в алфавите. Ключ — число сдвига.

Введите число в диапазоне 32–126, чтобы увидеть соответствующий ASCII-символ: —
Зачем это нужно: Все данные в компьютере хранятся в двоичной системе. Шестнадцатеричная используется в адресах памяти, цветах (#FF0000), MAC-адресах. Перевод — базовый навык программиста.
